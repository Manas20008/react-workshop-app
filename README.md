## ✨ Features  

- 🔐 **Secure Authentication** – JWT-based login & signup.  
- 👥 **Role-Based Access** – Roles: **Admin, Mentor, Student** with permissions.  
- 📅 **Workshop Management** – Organize workshops & assign mentors.  
- 🎨 **Clean UI** – Modern React-based frontend.
